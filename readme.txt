this is project (5) Neighborhood map
1.open index.html 
A list of places to Eat in Wallingford Ct will appear.
use the search box to narrow the amount of choices.
You can type in the Name of the Business or type in some of the menu 
items that are available to you. 
for example if you type in Mr or cupcakes it will display Mr D's 

from the list  you can click on a marker or on the business name and 
an information window will open giving you the street address and a menu choice.
It will also give a random Item with the USDA calories for that item .
for example if you are on Stella's pizza you might get calories for cheese pizza or pepperoni pizza.
